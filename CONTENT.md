# 如何編輯樂樂的回憶網站 (CONTENT.md)

這份文件會引導你如何替換網站上的文字與照片，讓這個紀念網站充滿屬於樂樂的真實回憶。

## 1. 準備素材
在開始編輯程式碼之前，建議先準備好以下照片。為了最佳視覺效果，請盡量符合建議的尺寸比例（裁切好後再放入）。

| 用途 | 檔名建議 (放在 `assets/img/`) | 建議比例 | 說明 |
| :--- | :--- | :--- | :--- |
| **首頁大圖 (Hero)** | `hero.jpg` | 16:9 或 3:2 | 這是打開網站看到的第一張大照片，建議選一張樂樂最燦爛的笑容或在大自然奔跑的照片。 |
| **時間軸照片** | `timeline-1.jpg`, `timeline-2.jpg`... | 4:3 或 1:1 (正方形) | 配合成長過程的關鍵時刻。 |
| **回憶牆照片** | `memory-1.jpg`, `memory-2.jpg`... | 1:1 (正方形) | 建議都裁切成正方形，或是全部統一步驟的 4:3，畫面會比較整齊療癒。 |

## 2. 如何修改文字與內容
所有的內容資料都集中在 `script.js` 這一個檔案中，你不需要去改 HTML 結構。

1.  用文字編輯器打開 `script.js`。
2.  找到 `const timelineEvents` 和 `const memories` 這兩個區塊。
3.  替換引號 `""` 內的文字。

### 修改時間軸 (Timeline)
```javascript
{
    date: "2010 春天", // 可以寫精確日期，也可以寫季節
    title: "初次見面",
    desc: "這一天，我們在收容所相遇...",
    img: "assets/img/timeline-1.jpg" // 你的照片路徑
}
```

### 修改回憶牆 (Memories)
```javascript
{
    title: "最愛的公園",
    tags: ["散步", "快樂"], // 標籤可用：散步, 睡覺, 搗蛋, 旅行, 撒嬌... (可自訂)
    desc: "每次來到這裡，樂樂總是興奮地轉圈圈。",
    img: "assets/img/memory-1.jpg"
}
```

## 3. 標籤 (Tags) 使用建議
標籤不僅幫助分類，也能快速回顧樂樂的不同面向。建議標籤：
- `日常` (Daily)
- `旅行` (Travel)
- `睡覺` (Sleep)
- `搞笑` (Funny)
- `生日` (Birthday)

## 4. 進階設定
如果你想要修改網站的主色調，可以打開 `style.css`，在最上方的 `:root` 區域修改顏色代碼：

```css
:root {
    --color-primary: #E0C097; /* 主色：溫暖的米棕色 */
    --color-bg: #FFFDF5;      /* 背景：奶油白 */
}
```
但請注意保持顏色的對比度，以免文字難以閱讀。
